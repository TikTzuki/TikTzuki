## Velero Backup (Kubernetes)

:::info
[Velero CLI Configuration Docs](https://velero.io/docs/v1.17/customize-installation/#optional-velero-cli-configurations)
:::

```shell
velero install \
  --provider aws \
  --plugins velero/velero-plugin-for-aws:v1.9.0 \
  --bucket velero-backup \
  --secret-file ./credentials-velero \
  --use-restic \
  --backup-location-config region=minio,s3ForcePathStyle="true",s3Url=http://minio-service.minio:9000
```
